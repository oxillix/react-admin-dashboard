Link to app: [https://admin-dasboard.netlify.app/](https://admin-dasboard.netlify.app/)

![Preview](preview.png)

### Instructions
1. Clone the project
```bash
git clone https://github.com/oxillix/react-admin-dashboard.git
```
2. Install npm packages
```bash
npm install
```
3. Start the app
```bash
npm start
```
4. (optional) create a production build
```bash
npm run build
```
